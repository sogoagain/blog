## 개발하기 🚀 

1. 환경 설정 파일

  - `properties.js`

2.  **테스트 코드 실행**

    ```shell
    npm run test
    npm run test:watch
    npm run test:coverage
    ```

3.  **개발 서버 실행**

    ```shell
    npm run start
    ```

4.  **prettier & lint**

    ```shell
    npm run format
    npm run lint
    ```

5.  **빌드하기**

    ```shell
    npm run build
    ```
